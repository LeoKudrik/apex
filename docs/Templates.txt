Mako Template:

To handle flash messages when using Mako, insert the following into your
templates:

<%namespace file="pyramid_apex:templates/flash_template.mako" import="*"/>
${apex_head()}

Insert this line where you would like the Flash Messages to be displayed:

${apex_flash()}

Jinja Templates:

{% import "pyramid_apex:templates/flash_template.jinja2" as flash -%}
{% block apex_head %}{% endblock -%}

Insert this line where you would like the Flash Messages to be displayed:

{% block apex_flash %}{% endblock -%}
